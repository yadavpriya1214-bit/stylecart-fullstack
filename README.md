# StyleCart - Full Stack E-Commerce Application

## Project Overview

StyleCart is a full-stack e-commerce web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to browse products, view product details, manage their shopping cart, place orders, and manage their accounts through a responsive and user-friendly interface.

---

## Features

### User Features

* User Registration and Login
* Product Browsing
* Product Details View
* Category-wise Product Filtering
* Add to Cart
* Remove from Cart
* Place Orders
* View Order History

### Admin Features

* View Customer Orders
* Manage Order Status
* Product Management

## Installation and Setup

### Clone Repository
